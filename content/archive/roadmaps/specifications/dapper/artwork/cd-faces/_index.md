# CD Faces

## Xubuntu cd face - generic

I like printing my CDs and I usually end editing images from Breezy (whose cd faces I like more than Dapper) with the GIMP.

I wanted something easier to scale so I've prepared a svg file. Everyone can easily add writings to it, like release number, "live|install" or whatever. 
This image is based on Breezy CD faces found at [DIYMarketing](https://wiki.ubuntu.com/DIYMarketing) and the logo found at [XubuntuArtwork](https://wiki.ubuntu.com/XubuntuArtwork) page.

![xubuntu_cd_face.png](/archive/roadmaps/specifications/dapper/artwork/cd-faces/xubuntu_cd_face.png)
