## Detailed specification

- List all greeters (parse .desktop-files in xgreeters)
- List all users, allow them to enable autologin, if they are sudoers, that is; use gksudo to apply changes.
- Allow users to specify timeout of their autologin.
- Allow users to pick desktop default session.
