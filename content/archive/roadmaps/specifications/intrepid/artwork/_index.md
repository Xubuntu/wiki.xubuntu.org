### Customized Murrina theme + daolo xfwm4 wm theme

![murrina-custom-daolo.png](/archive/roadmaps/specifications/intrepid/artwork/murrina-custom-daolo.png)
this is still a work in progress.  it features a customized murrina theme, and the tango-noir icon set.  the window manager theme is daolo, but i'm looking at others, as well.  just working toward something that looks clean, lightweight, and which uses colors that are complimentary to xubuntu.  -- j1mc
